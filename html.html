Röviden a HTML kódról

Mi is az a HTML?

A HTML egy leíró, vagy jelölő nyelv, ami a weboldalak struktúráját határozza meg. Olyan elemekből, vagy "tag"-ekből áll mint a p, img, table, h1, h2, h3... Ezek segítségével határozzuk meg, hogy a weboldalunknak milyen építőkövei legyenek.
A wikipédián itt tudtok a HTML-ről olvasni egy kicsit. Van egy külön oldal a tageknek is, ahol mindegyikről külön meg tudjátok nézni mit csinál azt pedig itt éritek el.

Egy kis CSS

Ha a HTML struktúrát, akkor a CSS (Cascading Style Sheets) adja a oldalanak a stílusát, vagyis a színeket, betűtípusokat, formázást, stb. (És a műkődést pedig a JS adja)

CSS-t meg lehet adni a HTML kódban, vagy külön egy .css fájlban is. Ha JS Bint használunk, a külön CSS fülön található kódot automatikusan ráhuzza a HTML-re, így olyan mintha külön fájlt használnánk. Nézzétek meg az ottani kódot, és írjátok át az értékeket.

A CSS-ben sok, de nem végtelen tulajdonság van. Ezeket főleg azoknak kell nagyon jól tudni, akik úgynevezett site-builddel foglalkoznak. Az ő feladatuk, hogy az oldal úgy nézzen ki minden böngészőben, ahogy a grafikusok megálmodták.

A canvas

Eddig Sanyi, a kód szörny segítségével használtuk a canvas HTML elemet, de most már ideje, hogy saját honlapot csináljunk. A következő kódrészlettel létre tudjuk hozni a saját rajzvászonunkat:

    
    <canvas id="canvas" width="400px" height="300px"></canvas>
    <script>
      var c = document.getElementById('canvas').getContext('2d');

      c.fillStyle = 'green';
      c.fillRect(10, 10, 100, 100);
    </script>
